# xCopyPath

It's similar to "Copy as path" but it's little bit more useful for programmers (at least for me);


# Usage: 

* Download [binaries](https://github.com/secrary/xCopyPath/releases)
* Copy binaries to desire folder, e.g `C:/Bin`
* Click `Setup.exe`
* That's all
